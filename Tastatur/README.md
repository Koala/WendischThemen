# Herausvorderungen beim Schreiben von Sonderzeichen mit Akut und Caron

Alles noch im Aufbau! :-)

## Vorwort

Das niedersorbische Alphabet basiert auf dem Lateinischen Alphabet, verwendet aber diakritische Zeichen wie den [Akut](https://de.wikipedia.org/wiki/Akut) und das [Caron (Kokolka)](https://de.wikipedia.org/wiki/Hatschek). Die Standardzeichenkodierung für das Sorbische Alphabet ist ISO 8859-2 (Latin-2).(Quelle: [Wikipedia](https://de.wikipedia.org/wiki/Niedersorbische_Sprache#Orthographie))  
Sprachcode nach ISO 639-3: dsb - Niedersorbisch / hsb - Obersorbisch  

Die Sonderzeichen:  
č, ć, ě, ł, ń, ŕ, š, ś, ž, ź, Č, Ł, Š, Ś, Ž, Ź  

Für das Schreiben dieser Zeichen gibt es je nach Betriebssystem diverse Lösungsmöglichkeiten.
